# Python-Simulator-For-Industrial-Automation-Modbus-and-IEC104-
Python GUI simulator for IEC 60870-5-104 (client/server) and Modbus TCP/RTU (master/slave). Upload Excel to configure signals, IPs, ports. Supports SP, DP, MV, coils, registers. Real-time logs, redundant 104, hybrid testing. Built with c104, pymodbus, Tkinter. Ideal for SCADA/ICS validation
